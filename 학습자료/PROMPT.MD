excel-automation이라는 이름의 엑셀 자동화 스킬을 만들어줘.
이 스킬은 사용자가 엑셀 파일을 전달하면 세 가지 작업을 수행해.

기능 1: 데이터 정리
- 중복 행 탐지 및 제거 (완전 일치 + 핵심 컬럼 기준 유사 중복)
- 전화번호 형식 통일 (010-XXXX-XXXX 형태로)
- 날짜 형식 통일 (YYYY-MM-DD 형태로)
- 빈 셀 탐지 및 처리 방법 제안
- 오타·이상값 탐지 (이메일 도메인 오타, 카테고리 오타 등)
- 정리 전후 변경 리포트 생성 (몇 건 수정, 어떤 항목이 변경되었는지)
- 결과를 새 엑셀 파일로 저장

기능 2: 분석과 시각화
- 데이터 요약 통계 (합계, 평균, 최대/최소, 추이)
- 사용자 질문에 맞는 분석 수행 ("지난달 대비 변화", "카테고리별 매출 비교" 등)
- 적절한 차트 자동 생성 (막대, 꺾은선, 파이, 히트맵 등)
- 핵심 인사이트 3~5개 텍스트로 요약
- 분석 결과를 새 시트 또는 새 파일로 저장

기능 3: 멀티탭 취합
- 여러 탭의 구조를 자동으로 파악
- 공통 키(상품코드, ID 등)를 기준으로 데이터 연결
- 각 탭에서 핵심 컬럼만 추출하여 하나의 관리 시트로 구성
- 탭마다 등록된 데이터 수가 다를 수 있음 (일부 탭에만 있는 상품 처리)
- 채널별/항목별 비교가 한눈에 가능한 레이아웃

실행 방식:
- 사용자가 자연어로 요청하면 세 기능 중 적절한 것을 자동 판단
- "정리해줘" → 기능 1 / "분석해줘" → 기능 2 / "취합해줘" → 기능 3
- 애매한 경우 사용자에게 어떤 작업을 원하는지 확인
- 결과물은 항상 엑셀 파일로 저장하고 변경 내역을 요약
